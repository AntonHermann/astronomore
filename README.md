# Astronomore

A wgpu-based graphics project with native and WebAssembly support.

## Commands

| Command | Description |
|---------|-------------|
| `just run` | Build and run natively |
| `just serve` | Build for WASM and start a local web server on port 8080 |
| `just build` | Build natively |
| `just build-wasm` | Build for WASM |
